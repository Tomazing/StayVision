StayVision Backend

This is the API server for StayVision, an interactive feature that helps potential guests simulate their stay before booking.

Available Endpoints

GET /api/health

Check if the API is running properly.

Example: GET https://stay-vision-backend.vercel.app/api/health

POST /api/getResponseFromLLM

Process a conversation with the LLM for a specific property.

Required payload:

{
  "propertyId": "wildhouse-farm",
  "messages": [],
  "userMessage": "Family of 4 with two teenagers. We love hiking and local food.",
  "systemPrompt": null
}

Example usage:

fetch('https://stay-vision-backend.vercel.app/api/getResponseFromLLM', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    propertyId: 'wildhouse-farm',
    messages: [],
    userMessage: "Family of 4 with two teenagers. We love hiking and local food."
  })
})

POST /api/feedback

Submit feedback about the simulation experience.

Required payload:

{
  "propertyId": "wildhouse-farm",
  "rating": 8,
  "feedback": "positive",
  "answers": {
    "initial": "Family of 4 with two teenagers. We love hiking and local food."
  }
}

Testing the API

Can test these endpoints using tools like:

  • Postman
  • cURL
  • Browser developer console
  • Any HTTP client that can make POST requests with JSON bodies

Available Properties

For demo purposes, the following property IDs are available:

  • wildhouse-farm - Wildhouse Farm in Milnrow
  • coastal-retreat - Coastal Retreat
  • mountain-lodge - Mountain Lodge
